## 3. Why multiple lenses

A single mean-variance number hides what an insurer actually manages: the same return can carry very different capital, very different earnings volatility, and a very different accounting (P&L vs OCI) footprint. The tables and charts in `outputs/` let those trade-offs be compared side by side.
