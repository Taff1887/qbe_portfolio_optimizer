## 2. Headline read across the lenses

- **Baseline**: 84% core fixed income / 16% risk assets; forward expected return **5.09%**, volatility **3.28%**, running carry **4.82%**, portfolio duration **4.2y**.
- **Mean-variance (lens 1)**: the Max-Sharpe portfolio lifts expected return to **5.08%** for volatility **1.70%** within the insurer constraints.
- **Stress (lens 2)**: worst instantaneous scenario is **Inflation / rates up** at **-11.26%** (P&L -9.87%, OCI -1.39%).
- **Capital (lens 5)**: binding LAGIC-style charge **2.54%** of assets (binding: Broad recession); return on capital **2.01x**.
- **Earnings (lens 3/6)**: annual earnings (P&L) volatility **5.72%**, chance of missing the 4.5% plan **12%**, carry funds **59%** of return.
- **Duration / ALM (lens 4)**: total dollar-duration gap **-0.29y**; a +100bp shock moves economic surplus **+0.29%** vs P&L earnings **+0.48%** (the gap is the OCI/surplus rate exposure, **-0.19%**).

## 3. Why multiple lenses

A single mean-variance number hides what an insurer actually manages: the same return can carry very different capital, very different earnings volatility, and a very different accounting (P&L vs OCI) footprint. The tables and charts in `outputs/` let those trade-offs be compared side by side.
